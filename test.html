Glossary

HTML

HyperText Markup Language is the standard markup language for documents designed to be displayed in a web browser.

CSS

Cascading Style Sheets is a style sheet language used for describing the presentation of a document written in HTML or XML.

JavaScript

JavaScript is a programming language commonly used in web development to create interactive effects within web browsers.

Operating System (OS) – نظام التشغيل

Software that manages computer hardware and software resources and provides common services for computer programs.
برمجيات تدير موارد الحاسوب المادية والبرمجية وتوفر خدمات عامة لبرامج الحاسوب.

User Interface (UI) – واجهة المستخدم

The means by which a user interacts with a computer system. Can be command-line (CLI), graphical (GUI), or batch.
الوسيلة التي يتفاعل بها المستخدم مع نظام الحاسوب، وقد تكون سطر أوامر، رسومية، أو دُفعية.

Command-Line Interface (CLI) – واجهة سطر الأوامر

A text-based user interface where commands are typed to interact with the system.
واجهة نصية يتم من خلالها إدخال الأوامر للتفاعل مع النظام.

Graphical User Interface (GUI) – الواجهة الرسومية للمستخدم

A user interface that uses visual elements like windows, icons, and menus for interaction.
واجهة مستخدم تستخدم عناصر بصرية مثل النوافذ والرموز والقوائم للتفاعل.

Program Execution – تنفيذ البرنامج

The process of loading a program into memory and running it.
عملية تحميل البرنامج في الذاكرة وتشغيله.

I/O Operations – عمليات الإدخال والإخراج

Input/Output operations, involving communication with devices or files.
عمليات تتعلق بالتواصل مع الأجهزة أو الملفات لإدخال البيانات أو إخراجها.

File-System Manipulation – معالجة نظام الملفات

Services related to managing files and directories, such as creating, deleting, reading, and writing.
خدمات متعلقة بإدارة الملفات والمجلدات مثل الإنشاء والحذف والقراءة والكتابة.

Communications – الاتصالات

Mechanisms for processes to exchange information, either within the same computer or across a network.
آليات تتيح للعمليات تبادل المعلومات سواء داخل نفس الحاسوب أو عبر الشبكة.

Error Detection – كشف الأخطاء

The process by which the OS identifies and handles errors in the CPU, memory, I/O devices, or user programs.
عملية يكتشف من خلالها نظام التشغيل الأخطاء ويتعامل معها في المعالج أو الذاكرة أو الأجهزة أو البرامج.

Resource Allocation – تخصيص الموارد

The process of assigning system resources (like CPU, memory, devices) to competing processes or users.
عملية تخصيص موارد النظام مثل المعالج والذاكرة والأجهزة للعمليات أو المستخدمين المتنافسين.

Accounting – المحاسبة

Tracking how much and what types of computer resources are used by different users or jobs.
تتبع كمية وأنواع الموارد الحاسوبية المستخدمة من قبل المستخدمين أو المهام المختلفة.

Protection – الحماية

Mechanisms to control access to system resources and prevent interference between concurrent processes.
آليات للتحكم في الوصول إلى موارد النظام ومنع التداخل بين العمليات المتزامنة.

Security – الأمان

Measures to protect the system from external threats and ensure the integrity and confidentiality of data, including user authentication.
إجراءات لحماية النظام من التهديدات الخارجية وضمان سلامة وسرية البيانات، بما في ذلك مصادقة المستخدم.

System Calls – نداءات النظام

Programming interface to the services provided by the OS, allowing user programs to request kernel actions.
واجهة برمجية لخدمات نظام التشغيل، تُمكّن البرامج من طلب تنفيذ أوامر من النواة.

API (Application Programming Interface) – واجهة برمجة التطبيقات

A set of functions that define how software components interact, often providing a higher-level abstraction for accessing OS services.
مجموعة من الدوال التي تحدد كيفية تفاعل مكونات البرمجيات، وتوفر غالباً تجريداً أعلى للوصول إلى خدمات النظام.

Monolithic Structure – البنية الأحادية

An OS design where the entire kernel is a single, large program.
تصميم لنظام التشغيل تكون فيه النواة برنامجاً كبيراً موحداً.

Layered Approach – النهج الطبقي

An OS design where the system is divided into a hierarchy of layers, each built upon lower layers.
تصميم يُقسَّم فيه النظام إلى طبقات هرمية، كل منها مبني على الطبقة التي تحتها.

Microkernel – النواة المصغرة

An OS design that provides only essential services in the kernel, with other services implemented as user-level processes.
تصميم يوفر فقط الخدمات الأساسية في النواة، بينما يتم تنفيذ بقية الخدمات كمكونات على مستوى المستخدم.

Modules – الوحدات البرمجية

An OS design where core services are in the kernel, and other services are implemented as loadable modules, allowing for flexibility.
تصميم يحتوي على الخدمات الأساسية في النواة، بينما يتم تنفيذ الخدمات الأخرى كوحدات قابلة للتحميل لتوفير المرونة.

Hybrid Systems – الأنظمة الهجينة

OS designs that combine elements of different structures, such as monolithic and modular.
تصاميم أنظمة تشغيل تجمع بين عناصر من هياكل مختلفة مثل الأحادية والوحداتية.

Kernel – النواة

The core part of the operating system that manages the system's resources.
الجزء الأساسي من نظام التشغيل المسؤول عن إدارة موارد النظام.
